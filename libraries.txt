Install these libraries in the Arduino IDE before compiling the project.

## Core Libraries / Dependencies

- Sensirion Core https://github.com/Sensirion/arduino-core/
- Sensirion arduino-i2c-sht3x https://github.com/Sensirion/arduino-i2c-sht3x
- RTClib https://github.com/adafruit/RTClib
- WiFiNINA https://docs.arduino.cc/libraries/wifinina/
- arduino-home-assistant by Dawid Chyrzynski https://github.com/dawidchyrzynski/arduino-home-assistant

## Additional Libraries

- AD5263 by Rob Tillaart https://github.com/RobTillaart/AD5263
  - For AD5263BRUZ50 dimmer control via I²C.
  - Note: The library is still in an early/test phase; project-specific adjustments may be required.
  - Project-specific additions or additional library functions may be required for the readback/plausibility checks required by the specification for fault detection.

- JC_EEPROM by Jack Christensen https://github.com/JChristensen/JC_EEPROM
  - Access to the external **AT24C32 EEPROM** of the
    WINGONEER Tiny DS3231 AT24C32 I2C module.
  - This library is fixed for the project.

- Streaming by Mikal Hart / Janelia https://github.com/janelia-arduino/Streaming
  - Required by **JC_EEPROM**.

## Intentionally No Longer Used

- SAMD_SafeFlashStorage
  - Persistence is no longer stored in the internal flash of the Nano 33 IoT.

## Notes

- Create local `Credentials.h` from `Credentials.example.h`
- Do not commit `Credentials.h`
- When changing RTC behavior, check availability of the required DS3231 alarm functions in RTClib:
  - `writeSqwPinMode(...)`
  - `setAlarm1(...)`
  - `setAlarm2(...)`
  - `alarmFired(...)`
  - `clearAlarm(...)`
